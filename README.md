## 실행 방법
1. 리포지토리를 클론합니다.
```
git clone https://github.com/David-Ha-1217/Inflearn_Java-Start.git
```
2. IntelliJ IDEA를 열고 프로젝트를 로드합니다.
3. Java 17이 프로젝트 SDK로 설정되어 있는지 확인한 후, 코드를 빌드하고 실행합니다.


## 환경 설정
- Windows11 24H2
- IntelliJ IDEA 2023.3.8
- Java 21.0.5
